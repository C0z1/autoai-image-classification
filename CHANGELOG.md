# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-16

### Added
- Initial release of AutoAI Image Classification project
- Feature extraction with ResNet50, MobileNetV2, EfficientNetB0
- Automated dataset creation for demonstration
- Complete Watson Studio AutoAI integration guide
- Comprehensive report template (2-4 pages)
- Data integrity checks (uniform dimensions, missing values, types)
- Batch processing for efficient feature extraction
- Sample visualization generation
- Documentation generator for feature extraction process
- Support for 3+ classes with 30-80 images
- Tabular dataset creation (CSV format)
- Requirements file with all dependencies
- MIT License
- Contributing guidelines
- Professional README with badges and quick start

### Documentation
- Complete Watson Studio setup guide with 19+ screenshot placeholders
- Step-by-step instructions for AutoAI experiment
- Results interpretation framework
- Feature importance analysis guide
- Overfitting/underfitting diagnosis methodology
- Pipeline comparison instructions
- Metric analysis guidelines
- Final report template with all sections

### Features
- Configurable CNN model selection
- Automatic sample dataset generation
- Image preprocessing and normalization
- Feature vector extraction (2048D for ResNet50)
- CSV export for Watson AutoAI
- Statistics and summary generation
- Visual sample creation
